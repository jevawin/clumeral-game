Convert to:
```html
<div class="min-h-screen bg-bg">
  <dialog data-fb-modal>...</dialog>
  <main class="max-w-sm mx-auto px-4">
    <header class="flex items-center gap-3.5 mb-2.5">
      <div data-octo-slot>
      <div data-octo-wrap title="click to replay" aria-hidden="true">
        ...octo SVG (unchanged)...

        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 218 43" fill="currentColor" aria-label="Clumeral" class="block">
```

Specific changes to each element:
1. `<div class="game">` --> `<div class="min-h-screen bg-bg">`
2. `<main class="game__inner">` --> `<main class="max-w-sm mx-auto px-4">`
3. `<header class="header">` --> `<header class="flex items-center gap-3.5 mb-2.5">`
4. `<div class="octo-slot" data-octo-slot>` --> `<div data-octo-slot>` (styling now handled by `[data-octo-slot]` in tailwind.css from Plan 01)
5. `<div class="octo" data-octo-wrap` --> `<div data-octo-wrap` (styling now handled by `[data-octo-wrap]` in tailwind.css from Plan 01)
6. `<h1 class="title">` --> `<h1 class="m-0 w-full text-text transition-colors duration-[400ms]">`
7. On the title SVG element, add `class="block"` (replaces `.title svg { display: block }`)
8. All `<g class="tlt">` elements: remove `class="tlt"` entirely (the .tlt class in style.css only sets cursor:default which is the SVG default anyway)
